Installation

Up and Running in Minutes

1
Download the Installer
Click the download button above. The installer includes Python, Whisper AI, and all required components.
2
Run the Installer
Run AnchorCastSetup_v1.2.0.exe. It installs Visual C++, copies Python, and creates Desktop shortcuts for AnchorCast and AnchorCast Timer.
3
Download the Speech Model
The installer offers to download the Whisper AI model (~244 MB). You can also do this later from inside the app.
4
Launch & Start Presenting
Open AnchorCast, connect your projector, and press F1 for the Getting Started guide. You're ready for Sunday.
